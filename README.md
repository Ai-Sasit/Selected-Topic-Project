# Selected-Topic-Project
## Project Description
  - Project name: Image Saving Website.
  - A sample website for back-up image.
  - ReactJS + SQLite + FastAPI
## Tools
  - SQLite
  - FastAPI
  - Docker
  - ReactJS
  - Swagger UI
  - SQLAlchemy
  - Uvicorn
  - Pydantic
## Start command
```
docker-compose up --build
```
## Stop command
```
docker-compose down
```
## Index Page
```
http://localhost:8000
```
## API URL - Swagger UI
```
http://localhost:8000/api/docs
```
